#Installation#

Installation of the accordion plugin is quite simple.

Include jQuery in your page if you did not already do so.  

```javascript
<script type="text/javascript">
/* define $ as jQuery just in case */
( function( $ )
{
	/* doc ready */
	$( function( )
	{
		$( '#my_accordion_1' ).accordion( );
	});
})( jQuery );
</script>
```
